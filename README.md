## Description

This project analyzes employee mental health data and its relationship with workplace productivity. Using a Streamlit-based interactive dashboard, users can explore key insights through visualizations, geographic mappings, and predictive analytics to understand how mental health factors impact employee performance.

## How to Clone

```bash
# Clone the repository
git clone https://github.com/Mariana-Gutierrez-R/Seguimiento_2_Datos.git

# Navigate to the project directory
cd Seguimiento_2_Datos

# Install dependencies
pip install -r requirements.txt

# Run the Streamlit application
streamlit run app.py
```

## Features

The Streamlit application includes the following features:

- **Data Overview** - Comprehensive view of the dataset with key statistics and summary
- **Descriptive Graphics** - Interactive visualizations using Plotly for data exploration
- **Geographic Map** - Spatial visualization showing data distribution by location
- **Frequency Tables** - Detailed tabulated data for categorical variables
- **Filters** - Interactive filtering options to refine data analysis
- **Predictive Analysis** - Machine learning models using Scikit-learn for outcome predictions

## Dataset Information

The dataset contains information about employee mental health and productivity metrics. Variables include:

- Employee demographics (age, gender, location)
- Mental health indicators (stress levels, work-life balance, mental health history)
- Productivity metrics (performance ratings, absenteeism, overtime hours)
- Workplace factors (company size, industry, job role)

**Source:** [Kaggle - Mental Health Dataset](https://www.kaggle.com/datasets)
